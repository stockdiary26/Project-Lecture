## WHAT — 어떤 문제를 푸는가

### 핵심 문제

학부모 상담 준비의 **전 파이프라인**(점수 정리 → 분석·비교 → 설명문 작성)이 수작업이다. 한 구간만 자동화해서는 체감이 없고, 전체를 줄여야 의미가 있다.

### 기존 도구가 실패하는 이유

- 학원 관리 앱: 학원 운영(수강료·출결·스케줄) 중심이라 번거롭고, 강사의 **상담 준비** 맥락에 맞지 않음
- 엑셀/스프레드시트: 분석까지는 가능하나 **문장 작성(가장 힘든 부분)**은 여전히 수동
- 공통 문제: 강사가 실제로 하는 "숫자 → 이야기" 전환을 지원하지 않음

## HOW — 어떻게 해결하는가

### 솔루션: 성적 대시보드 + AI 상담문 생성

점수를 입력하면 자동 분석(변화 추이, 반 평균 대비, 영역별 강약점) 후 AI가 학부모용 상담 설명문 초안을 생성한다.

### 플랫폼

웹 앱 (Streamlit Cloud 배포, 브라우저로 접속. 남편 PC에서 북마크로 진입)

### 사용 대상

1인 사용 (강사 본인 전용)

### MVP 기능 목록 (MoSCoW)

| 분류 | 기능 | 설명 |
|------|------|------|
| **Must** | 학생 정보 등록 | 반·학생명·학교명 기본 관리 |
| **Must** | 시험 점수 입력 | 시험별·학생별 점수 입력 (총점, 내신 등급, 영역별 세부 점수) |
| **Must** | 시험 유형 구분 | 학교시험 / 모의고사 별도 관리 및 비교 |
| **Must** | 영역 카테고리 관리 | 점수 영역(문법/독해/문학 등)을 추가·삭제·수정 가능. 강사가 자유롭게 커스터마이징 |
| **Must** | 성적 분석 자동화 | 이전 시험 대비 총점+영역별 변화, 반 평균 대비 위치, 영역별 강약점 자동 계산 |
| **Must** | 성적 추이 차트 | 학생별 시험 회차별 점수 변화를 시각적 그래프로 표시 |
| **Must** | AI 상담문 생성 | 분석 결과를 바탕으로 학부모용 설명문 초안 자동 생성 |
| **Must** | 학생별 메모 | 상담 이후 특이사항·후속 조치 기록 |
| **Should** | 상담문 편집·저장 | AI 초안을 교사가 수정하고 저장 |
| **Should** | 데이터 내보내기 | 분석 결과·상담문을 한글/PDF/엑셀로 내보내기 |
| **Could** | 반 전체 통계 대시보드 | 반별 평균·분포·등급 컷 시각화 |
| **Won't** | 학부모 직접 전달(알림/메일) | 이번 버전에서는 제외 |
| **Won't** | 다중 교사/학교 지원 | 1인 사용이므로 제외 |
| **Won't** | 모바일 앱 | 집 PC 사용이므로 제외 |

### 목표 효과

학생당 상담 준비 시간: **20분 → 2~3분**
70명 기준 총 소요: **18시간 → 약 3시간**
연간 절약: **약 120시간**

## 제외 범위

- 학부모에게 직접 알림·메일 발송하는 기능
- 복수 강사 간 데이터 공유·협업 기능
- 모바일 앱 (반응형 웹은 고려하되 전용 앱은 만들지 않음)
- 출결·수강료 등 학원 관리 기능
- 학교 시스템(NEIS) 연동 (학생이 성적표를 가져오면 강사가 수동 입력)
