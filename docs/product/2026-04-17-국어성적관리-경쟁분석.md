# 국어 성적 관리 & 상담 준비 도구 — 경쟁 분석

## 상세 분석

### 1. 학원 관리 앱 (학원별, 하이클래스, 셔틀 등)

**강점:**
- 수강생 관리, 출결, 수강료 청구 등 학원 운영 전반 커버
- 학부모 앱 연동으로 알림·공지 가능
- 이미 학원 시장에 자리잡은 서비스

**구조적 한계:**
- **운영 도구이지 교육 도구가 아님** — 수강료·출결·스케줄이 핵심, 성적 분석은 부가 기능
- 성적 입력은 가능하나 **학교 시험·모의고사 영역별 세부 분석 부재**
- 학부모 상담 준비에 필요한 **"이 학생이 왜 이렇게 변했고, 앞으로 뭘 해야 하는가"** 서사를 만들어주지 않음
- 기능이 많아 **번거롭다** — 남편분이 안 쓰게 된 이유

**우리가 이기는 지점:** 학원 운영 기능을 전부 빼고, 오직 **점수 → 분석 → 상담문** 파이프라인에만 집중. 단순함 자체가 차별점.

### 2. 엑셀 / 구글시트

**강점:**
- 자유도 최고 — 원하는 형태로 표·차트 구성 가능
- 추가 비용 없음
- 익숙한 도구

**구조적 한계:**
- **매 시험마다 수식·차트를 다시 세팅**하거나 템플릿을 관리해야 함
- 70명분 반복 작업의 자동화가 어려움 (매크로 작성 능력 필요)
- **상담문 생성 기능 없음** — 분석까지는 가능하나 "숫자 → 이야기" 구간은 여전히 수동

**우리가 이기는 지점:** 세팅 없이 점수만 넣으면 분석 + 상담문까지 한 흐름. 반복 세팅 비용 제로.

### 3. 범용 AI (ChatGPT, Claude 직접 사용)

**강점:**
- 문장 생성 능력 우수
- 무료 또는 저렴한 구독료

**구조적 한계:**
- **데이터가 누적되지 않음** — 매번 이전 시험 점수를 복붙해야 함
- 이전 상담 내용·메모와 연결 불가
- 점수 데이터 형식을 매번 프롬프트로 설명해야 하는 번거로움
- 학생 70명을 하나씩 돌리면 오히려 시간 소모

**우리가 이기는 지점:** 데이터를 누적 관리하면서 AI를 내장. "점수 입력 → 이전 이력 자동 참조 → 상담문 생성"이 끊김 없이 연결.

## 핵심 차별화 요약

> 기존 도구들은 "성적을 **저장**하는 것"까지는 하지만, "성적을 **이야기로 바꾸는 것**"은 하지 않는다. 우리 제품은 이 마지막 구간(AI 상담문 생성)을 핵심으로 두고, 앞 단계(입력·분석)를 그에 최적화된 형태로 붙인다.

## 리스크

- 학원 관리 앱이 AI 상담문 기능을 추가하면 경쟁력 약화 → **속도가 방어선**. 1인 강사에게 먼저 검증하고 빠르게 반복.
- 범용 AI 성능 향상으로 프롬프트만으로도 충분해지면 → **데이터 누적(이력 관리)**이 방어선. 단발 AI 호출로는 추이·이전 상담 맥락을 활용할 수 없음.
