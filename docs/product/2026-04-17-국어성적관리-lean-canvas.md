# 국어 성적 관리 & 상담 준비 도구 — Lean Canvas

| 칸 | 내용 |
|----|------|
| **문제** | 1. 학부모 상담 준비에 학생당 20분 × 70명 = 18시간 소요 (연 144시간+) |
|  | 2. 점수 정리 → 분석 → 상담문 작성 전 과정이 한글문서 수작업 |
|  | 3. 기존 학원 관리 앱은 운영(수강료·출결) 중심이라 번거롭고, 강사의 "숫자 → 이야기" 전환을 지원하지 않음 |
| **고객 세그먼트** | 국어학원 강사. 7개 반 70명 이상 담당. 학생들의 학교 시험·모의고사 후 학부모 상담을 정기적으로 준비하는 강사. 현재 1인 사용(남편) 대상. |
| **고유 가치 제안** | 점수 입력 한 번으로 성적 분석부터 학부모 상담문 초안까지 — 18시간을 3시간으로 줄이는 학원 강사 전용 도구 |
| **솔루션** | 1. 시험별 점수 입력 + 자동 분석 (변화·평균 대비·영역별 강약점) |
|  | 2. 성적 추이 차트로 시각적 파악 |
|  | 3. AI가 분석 결과를 학부모용 상담 설명문 초안으로 변환 |
| **채널** | 첫 사용자는 남편(직접 전달). 이후 확장 시: 학원 강사 커뮤니티, 국어과 교사·강사 단톡방, 학원 연합회 입소문 |
| **수익 모델** | v1은 개인 프로젝트(수익화 없음). 향후 확장 시: AI 상담문 생성 횟수 기반 프리미엄 구독 또는 학원 단위 라이선스 |
| **비용 구조** | AI API 호출 비용 (Claude/GPT), 웹 호스팅 비용, 개발 인건비(본인 시간) |
| **핵심 지표** | 1. 학생당 상담 준비 시간: 목표 **3분 이하** (현재 20분) |
|  | 2. 상담문 AI 초안 채택률: 목표 **70% 이상** (강사가 초안을 크게 수정 없이 사용하는 비율) |
| **경쟁 우위** | 학원 강사의 실제 워크플로우(점수 → 분석 → 상담문)에 맞춘 end-to-end 파이프라인. 학원 관리 앱이나 범용 스프레드시트가 커버하지 못하는 "숫자를 이야기로 바꾸는" 마지막 구간을 AI로 해결. |
