# Changelog

## [0.1.0] - 2026-04-23

국어학원 성적 관리 + AI 상담문 생성 앱 최초 공개 버전.

### 새 기능

- **로그인과 대시보드 홈**: 강사 계정으로 접속해 담당 학급/학생 현황을 한눈에 확인.
- **학생 관리**: 학급·학교별 뷰에서 학생을 조회하고, CSV 파일로 명단을 일괄 업로드. 2단계 확인을 거쳐 안전하게 학생을 삭제.
- **카테고리 설정**: 문법·독해·작문 등 채점 카테고리를 자유롭게 추가·수정.
- **성적 입력**: 시험을 만들고 학생별 탭과 시험별 탭에서 같은 데이터를 양쪽 방향으로 입력. 데이터 편집기로 대량 수정 가능.
- **성적 분석**: 학생 선택 시 시간에 따른 총점·카테고리별 추세 그래프와 다른 학생·학급 평균과의 비교 차트 제공. 학생 목록에서 행을 클릭하면 해당 학생 분석 페이지로 바로 이동.
- **학생별 메모**: 상담 준비에 쓰는 관찰 기록을 학생 페이지에서 작성하고, 기존 메모를 인라인으로 바로 수정.
- **AI 상담문 생성**: 성적과 메모를 근거로 Claude가 학부모용 상담 초안을 작성. 강사가 내용을 편집하거나 톤을 바꿔 재생성 가능.

### 개선

- 동일 날짜에 치른 서로 다른 시험이 추세 차트에서 겹쳐 보이던 문제를 해결해 각각 분리 표시.
- 학생 간 입력 화면 이동 시 이전 학생의 카테고리 점수가 잔존하던 문제 수정.
